# Brent Tracker

Software per il monitoraggio del prezzo del petrolio Brent.

## Requisiti

* [Docker](https://www.docker.com/)

## Avvio dell'applicazione

Per avviare il sistema, eseguire il seguente comando da terminale:

```bash
docker run -p 8000:8000 matteoandriolo00/brent-tracker:latest
```

e aprire il link fornito da Docker.
