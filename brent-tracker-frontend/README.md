# Brent Tracker Frontend

Frontend React + Vite per il backend FastAPI Brent Tracker.

## Requisiti

- Node.js
- Backend FastAPI attivo su `http://127.0.0.1:8000`

## Avvio

```bash
npm install
npm run dev
```

Poi apri l'indirizzo mostrato da Vite, normalmente:

http://localhost:5173

## Nota CORS

Se il browser blocca le richieste al backend per CORS, aggiungi il middleware CORS a FastAPI.
